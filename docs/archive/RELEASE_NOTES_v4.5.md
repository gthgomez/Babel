# HISTORICAL / UNSUPPORTED — OLS Compiler v4.5 Release Notes

> This document is retained only as a historical record. The former v4.5
> release announcement described aspirational compiler modules and performance
> claims that are not supported by verified public implementation evidence. It
> is not a current release note or compatibility statement.

The archived announcement covered proposed signature-based optimization,
dynamic guideline selection, multi-agent routing patterns, and compiler
self-application controls. Those descriptions must not be treated as available
public features.

Current capabilities are documented in [`../../README.md`](../../README.md) and
the CLI evidence boundaries are documented in
[`../../babel-cli/README.md`](../../babel-cli/README.md).

Future release notes must link each material capability claim to the shipped
implementation and a reproducible test or validation command.
