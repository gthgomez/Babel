# Babel v0.1 Foundation

Babel is a control plane for AI work. It automatically chooses the right instruction stack for a task so the model follows the correct rules, domain guidance, and project context instead of improvising from a single prompt.

## What Is New In This Release

- A clearer onboarding path with `START_HERE.md`
- A tighter `README.md` focused on first understanding and first use
- A lightweight `examples/first-success.md` showing why Babel is useful right away
- A cleaner public story with `v9` as the visible default routing lane

## What Babel Does

Babel reads the task, selects the right instruction layers, resolves them into a compiled prompt stack, and hands that stack to the model before work begins.

## How To Start

1. Open `START_HERE.md`
2. Read `README.md`
3. Try the example in `examples/first-success.md`
4. Use Babel with a request like: `Use Babel to build X`

## What This Release Focuses On

- Fast comprehension for a new user
- Easier first successful use
- Clear public positioning around the `v9` routing lane
- A stable foundation for immediate OSS release follow-up

## Routing Note

`v9` is the public default. `v8` remains in the runtime only as a compatibility fallback.
