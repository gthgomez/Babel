# Example Mobile Finance Context

Use this project overlay for a local-first mobile finance demo. It assumes a Kotlin/Compose client, a small persisted data model, and strict care around currency values, user-visible state, and offline recovery.

Public expectations:
- Keep examples generic and reproducible.
- Use placeholder package names and sample data only.
- Treat financial calculations as deterministic and testable.
- Prefer clear onboarding commands over private workspace assumptions.
