# Example Game Suite Context

Use this project overlay for a small tower-defense or action-strategy game example. It focuses on game-state invariants, enemy waves, upgrade paths, and portable project structure.

Public expectations:
- Keep gameplay rules deterministic enough to test.
- Treat save data, input, audio, and UI as separate surfaces.
- Include smoke evidence for changed scenes or scripts.
- Avoid private prototype names, paths, or local machine references.
