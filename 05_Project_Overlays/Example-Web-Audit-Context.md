# Project Overlay — Example Web Audit

## Purpose

Sanitized example overlay for a web auditing and compliance reporting product.

## Example Stack

- web app frontend
- structured report generation
- persisted evidence artifacts
- validation and remediation workflows

## Hard Invariants

- audit records are append-only
- evidence must stay attributable and reproducible
- remediation guidance must remain specific and testable
- compliance reporting must prefer clarity over marketing copy

## Primary Objects

- audit entries
- evidence artifacts
- reports
- remediation items
