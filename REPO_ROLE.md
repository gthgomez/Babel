# Repo Role

This repo is `Babel-public`, the public-facing sanitized export of Babel.

Use this repo to publish Babel's reusable control plane, onboarding flow, example stacks, and public-safe runtime surfaces.

Do not store private repo names, local paths, package IDs, SKUs, deployment URLs, operator notes, or private heuristics here.

`v9` is the visible default product story. `v8` remains compatibility-only in the background.

This repo is derived. The source-of-truth lives in `Babel-private`.
